# 📚 Library Management System (Flask + MySQL)

## 📌 Project Overview

This is a full-stack **Library Management System** built using:
- 🐍 Flask (Python Backend)
- 🛢️ MySQL (Database)
- 🌐 HTML (Frontend)
- 🎨 CSS (Styling)

It helps manage library operations like:
- User Login & Signup
- Add / View Books
- Issue Books
- Return Books
- Fine Calculation System

---

## ⚙️ Tech Stack

- 🐍 Python (Flask)
- 🛢️ MySQL
- 🌐 HTML
- 🎨 CSS
- 🔗 Flask-MySQL Connector

---

## ✨ Features

### 🔐 Authentication
- Signup system
- Login system

### 📚 Book Management
- Add books
- View books
- Track availability

### 🔄 Issue System
- Issue books to users
- Mark as unavailable

### 📥 Return System
- Return books
- Update status

### 💰 Fine System
- ₹10 per day after 7 days
- Auto fine calculation

## 🚀 Installation

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/library-management-system.git
cd library-management-system

### 2️⃣ Install Dependencies
pip install flask mysql-connector-python

### 3️⃣ Setup Database
mysql -u root -p

Then create database and tables using schema provided in project.

### 4️⃣ Run Project
python app.py

### 5️⃣ Open Browser
http://127.0.0.1:5000
